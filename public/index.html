4 lines. One POST. Any agent.

Moodring is a stateless function call. No auth, no sessions, no API keys. Just x402 payment + JSON in, JSON out.

cURL
const res = await fetch('https://moodring.onrender.com/api/mood/read', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ text: 'Just lost 30% on a position. Sleep is going to be tough.' }),
});
// Returns 402 with PAYMENT-REQUIRED header
// Sign with your EIP-3009 wallet, retry with PAYMENT-SIGNATURE header, get JSON back
import requests
r = requests.post('https://moodring.onrender.com/api/mood/read', json={
    'text': 'Just lost 30% on a position. Sleep is going to be tough.'
})
